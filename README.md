# Stump

这是一个提供可视化的文件管理工具，可以帮助你快速检索你需要的文件。

## 安装

```bash
$ npm install -g @jokerwon/stump
```

## 使用

```bash
$ stump src/assets
```

## 功能

### 已实现

- [x] 可视化界面
- [x] 检索指定目录的 `.png` 文件
- [x] 快速复制路径
- [x] 自动寻找可用端口
